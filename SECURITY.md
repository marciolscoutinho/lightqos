# Security Policy

## Supported Versions

| Version | Support                  |
| ------- | ------------------------ |
| 0.2.x   | ✅ Active                 |
| 0.1.x   | ⚠️ Critical patches only |
| < 0.1   | ❌ No support             |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Send an email to: marciolscoutinho@gmail.com

Include:

- Vulnerability description
- Steps to reproduce
- Potential impact
- Affected versions

We will respond within **48/60 hours** with confirmation and a remediation plan.
